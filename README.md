# EcoRuta 

Claudia Urbano 

Considero que este proyecto puede aportar mucho a la comunidad, ya que muchas personas no conocen los horarios del carro recolector, 
lo que genera acumulación de basura y contaminación en los barrios. Con esta plataforma buscamos que la información sea más accesible 
y que los ciudadanos puedan organizar mejor la disposición de sus residuos.

Además, el sistema permitirá reportar quejas y reclamos, fortaleciendo la comunicación entre la comunidad y la empresa encargada del servicio. 
De esta manera, se podrá brindar una atención más rápida, organizada y eficiente para beneficio de todos.
